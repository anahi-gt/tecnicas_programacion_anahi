# Técnicas de Programación – Comparación Tradicional y POO

## Descripción
Este repositorio contiene un programa en Python que calcula el promedio semanal del clima utilizando dos enfoques de programación: Programación Tradicional y Programación Orientada a Objetos (POO).

## Programación Tradicional
En la programación tradicional se utilizan funciones para ingresar las temperaturas de los días de la semana y calcular el promedio semanal. El código sigue una estructura secuencial y sencilla.

## Programación Orientada a Objetos (POO)
En la programación orientada a objetos se utiliza una clase para encapsular los datos del clima y los métodos necesarios para ingresar las temperaturas y calcular el promedio semanal.

## Comparación
Ambos enfoques permiten resolver el mismo problema. La Programación Tradicional es más simple y directa, ideal para programas pequeños. Por otro lado, la Programación Orientada a Objetos ofrece una mejor organización del código, facilitando su mantenimiento y reutilización.
