Ejemplos del Mundo Real – Programación Orientada a Objetos

# Descripción del proyecto
Este proyecto presenta un ejemplo del mundo real desarrollado en Python
utilizando Programación Orientada a Objetos (POO).

El caso implementado corresponde a un sistema de una tienda en línea,
donde se modelan objetos reales como productos, clientes y la tienda,
demostrando la interacción entre clases, atributos y métodos.

# Objetivo
Aplicar los conceptos fundamentales de la Programación Orientada a Objetos
mediante un ejemplo práctico del mundo real.
